Note, plain zip archives can be validated by disabled validation of OMEX Manifests. Note, SED-ML validation is required to validate models. Note, model validation is required to comprehensively validate SED-ML files, including validating the targets to models for model changes and simulation observables. Note, BioSimulations validates models using community-contributed validation tools for each model language. The thoroughness of these tools varies. In addition, some tools currently provide limited diagnostic information. We welcome contributions of improved validation tools. Note, projects published to BioSimulations must contain valid metadata that meets BioSimulations minimum metadata conventions. Note, an OMEX manifest file is required to validate metadata files. As a result, OMEX Metadata validation should generally be disabled when OMEX manifest validation is disabled. Note, image validation is required to comprehensively validate thumbnail images described in OMEX Metadata files.
